About This Website

This website is maintained by the Apache Community Development PMC who communicate on the dev@community.apache.org archived mailing list.

The text for this website is static HTML, but most of the work happens in JavaScript which loads a variety of JSON data sources to display the project report sample.
The underlying sources are stored in a Subversion repository at: 

svn co https://svn.apache.org/repos/asf/comdev/reporter.apache.org/trunk/

Changes checked into the HTML are reflected immediately on the production server.

Bugs with the reporter tool can be reported in JIRA's COMDEV project, component Reporter.

See README.txt for technical details on how the site builds the data.
